# Minimum Viable Relationships (MVR) Framework V3 Archive

This repository preserves the indexed V3 research record for the Minimum Viable Relationships (MVR) framework by Farouk Mark Mukiibi and African Market OS.

It is intentionally kept online for DOI, citation, and historical discoverability. It is not the canonical runtime contract for the current MVR API.

## Current Canonical API

Use these current public API surfaces instead of older V3 API examples:

- Agent OpenAPI: https://africanmarketos.com/api/openapi.agent.json
- Full OpenAPI: https://africanmarketos.com/api/openapi.json
- OpenAPI YAML compatibility path: https://africanmarketos.com/api/openapi.yaml
- API docs: https://africanmarketos.com/api/docs
- Public sandbox guide: https://africanmarketos.com/docs/sandbox.md
- Agent instructions: https://africanmarketos.com/AGENTS.md
- LLM discovery file: https://africanmarketos.com/llms.txt
- MCP endpoint: https://africanmarketos.com/mcp
- MCP registry metadata: https://github.com/africanmarketos591/mvr-framework/blob/main/server.json

For non-commercial evaluation of the current MVR Core API v6.32.0, use:

```text
X-API-Key: mvr-demo-key-2026
```

The public sandbox is rate-limited, locked to `full_advisory` and `client_safe` output, and is not for production, credit scoring, legal certification, model training, reverse engineering, or commercial resale.

## Why This Repository Still Exists

Older research versions, citation files, and DOI-linked materials can remain valuable for academic provenance and search indexing. To prevent AI agents and developers from generating calls against stale API shapes, the OpenAPI files in this repository now point to the current MVR API agent contract.
